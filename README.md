# movie-web-record-collection
该仓库主要用于收集和记录影视相关网站的公开基础记录信息，涵盖电影、电视剧及综合影视内容站点。内容仅限站点层级的客观资料，不包含任何影视内容文件或播放链接。所有资料整理自公开渠道，目的在于形成结构化、可查询的站点信息记录集合，便于后续对照与查阅。本项目不参与站点运营，也不对任何网站作主观评价，仅作为信息整理与长期存档用途存在。
